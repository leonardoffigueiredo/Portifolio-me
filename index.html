Trabalhos Realizados

Aumentar Vocabulário

2023

HTML | CSS | JAVASCRIPT | PHP

Neste projeto fiz uma ferramenta online que sugere palavras mais ricas para substituir aquelas comuns usadas diariamente, visando enriquecer o vocabulário dos usuários e tornar suas conversas mais interessantes e variadas. Os usuários podem acessar essa funcionalidade facilmente através de um navegador web padrão, e o site foi desenvolvido com HTML, CSS e JavaScript para proporcionar uma experiência interativa e amigável.

Galeria de Fotos Rotativa com CSS

2023

HTML | CSS

Destaco um projeto pioneiro de galeria rotativa concebida exclusivamente por meio do CSS. Este inovador projeto entrega uma experiência interativa imersiva, desprovida de dependências externas, apoiando-se unicamente em técnicas avançadas de estilização com CSS. A galeria rotativa é um verdadeiro expoente do poder criativo do CSS, alavancando animações, transformações e transições para oferecer aos usuários uma imersão completa na exploração de uma coleção de elementos.

Site responsivo de músicas

2023

HTML | CSS | JAVASCRIPT

Neste projeto, desenvolvo um site altamente responsivo utilizando as linguagens HTML, CSS e JavaScript. Trata-se de uma simulação de uma plataforma de música online, onde são apresentados reprodutores fictícios para as faixas musicais. Através da combinação de técnicas de design e programação, o site oferece uma experiência imersiva, permitindo aos usuários explorar e interagir com a interface de reprodução de música de forma intuitiva e envolvente.

Site de Exposição Para Vendas de Carros

2024

HTML | CSS | JAVASCRIPT

Um lugar onde você pode encontrar uma seleção de veículos para explorar. Ao navegar pelo site, você verá várias imagens dos carros disponíveis. Quando encontrar um veículo que lhe interesse, basta clicar na imagem e você será direcionado para o site onde o carro está anunciado, para que possa obter mais informações e até mesmo fazer a compra, se desejar. É uma maneira conveniente de visualizar diferentes opções de veículos e encontrar aquele que melhor atende às suas necessidades e preferências. O site foi desenvolvido com as linguagens: HTML, CSS e JavaScript.
